## Kivy Tic Tac Toe

Its a Tic Tac Toe game made with Kivy. The computer moves works with the help of `Minimax` algorithm.

![screen](img/Capture.PNG)

### Kivy Installation

```
python -m pip install kivy[base] kivy_examples
```
